# POLAREBIKES (polarebikes.com)

E-commerce site for **Polarebikes LLC** — Next.js App Router, Tailwind CSS, shadcn/ui-style components, Framer Motion, and Zustand (cart with localStorage persistence).

## Requirements

- Node.js 20+
- [pnpm](https://pnpm.io/) 9+

## Setup

```bash
pnpm install
```

## Scripts

| Command        | Description              |
|----------------|--------------------------|
| `pnpm dev`     | Dev server (Turbopack)   |
| `pnpm build`   | Production build         |
| `pnpm start`   | Run production server    |
| `pnpm lint`    | ESLint                   |

## Local preview

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Radix UI / shadcn patterns
- Lucide React, Framer Motion, Zustand
